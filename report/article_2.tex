\begin{abstract}
  In this report, we will explain our two solutions for implementing Conway's Game of Life in golang. The first solution is designed to be run on a
  single machine, and makes use of all the cpu threads available on a computer through the parralelization of the program. The second solution is designed
  to run on multiple machines, allowing for the work to be distributed accross multiple machines. This second solution uses aspects of the first solution
  within each worker.
\end{abstract}
}
